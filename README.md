# Tingi Petroleum Services Website

A modern, responsive website for Tingi Petroleum Services, a leading clean fuel supplier in South Africa. Built with clean HTML, CSS, and JavaScript, this website showcases the company's fuel supply services, depot network, and sustainability initiatives.

## 🏢 About Tingi Petroleum Services

Tingi Financial Services (Pty) Ltd, trading as Tingi Petroleum Services, is a Midrand-based fuel supply company that provides premium 50ppm clean diesel to industrial, commercial, mining, logistics, and agricultural operations across South Africa.

### Key Features
- **National Coverage**: 7 depots across South Africa
- **Premium Fuel**: 50ppm clean diesel supply
- **24/7 Service**: Round-the-clock supply capability
- **Compliance**: B-BBEE Level 1, HSE Compliant, CIPC Registered
- **Multiple Services**: Bulk diesel, mobile refueling, tanks & equipment, product testing

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Navigate to the project directory
3. Open `index.html` in your browser
4. For development, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

### Configuration
- Update contact information in `index.html` and `contact.html`
- Modify form submission email address
- Update depot information as needed
- Customize branding assets in the `images/` folder
